Bibliotecas Python Mais Usadas

Python possui milhares de bibliotecas para acelerar o desenvolvimento. Aqui estão as mais populares para iniciantes e profissionais, detalhando como importar, para que servem e exemplos de uso comentados.

random

Para que serve?
Geração de números aleatórios, sorteios, escolhas aleatórias em listas e embaralhamento de itens. Essencial para jogos, simulações e testes.

Como importar: 
import random
Exemplo: 
# Número inteiro aleatório de 1 a 10
print(random.randint(1, 10))

# Escolhe um elemento aleatório de uma lista
print(random.choice(['a', 'b', 'c']))

# Embaralha uma lista (altera a lista original)
lista = [1, 2, 3, 4]
random.shuffle(lista)
print(lista)

collections

Para que serve?
Tipos de dados avançados: dicionários com valor padrão, contadores de itens, filas (deque), pilhas e mais.

Como importar:

Exemplo:

Para que serve?
Funções para criar e manipular iteradores, combinações, permutações e ciclos infinitos de forma eficiente.

Como importar:

Exemplo:

functools

Para que serve?
Ferramentas para programação funcional: acumuladores, decoradores, cache de funções e mais.

Como importar:

Exemplo:

# Cache automático para funções recursivas (exemplo: Fibonacci)

print([fib(i) for i in range(7)])

Desafio Final: Projeto Multi-bibliotecas

Desafio: Monte um script profissional que:

  • Lê um arquivo CSV com nomes e datas de nascimento
  • Calcula idade de cada pessoa e sorteia um nome aleatório
  • Gera um gráfico de quantas pessoas nasceram em cada mês
  • Salva um arquivo JSON com todos os resultados
  • Deve ser modular, usar funções e bibliotecas padrão e externas (csv, random, datetime, json, matplotlib)
Solução sugerida (comentada): 
import csv
import random
from datetime import datetime
import json
import matplotlib.pyplot as plt

def ler_csv(arquivo):
    """Lê CSV e retorna lista de pessoas (nome, data_nasc)"""
    pessoas = []
    with open(arquivo, encoding="utf-8") as f:
        reader = csv.DictReader(f)
        for row in reader:
            pessoas.append({"nome": row["nome"], "data_nasc": row["data_nasc"]})
    return pessoas

def calcular_idades(pessoas):
    """Calcula idade de cada pessoa"""
    hoje = datetime.now()
    for p in pessoas:
        nasc = datetime.strptime(p["data_nasc"], "%Y-%m-%d")
        p["idade"] = hoje.year - nasc.year - ((hoje.month, hoje.day) < (nasc.month, nasc.day))
    return pessoas

def sorteio_nome(pessoas):
    """Sorteia um nome aleatório"""
    return random.choice(pessoas)["nome"]

def grafico_nascimentos(pessoas):
    """Gera gráfico de nascimentos por mês"""
    meses = [0]*12
    for p in pessoas:
        nasc = datetime.strptime(p["data_nasc"], "%Y-%m-%d")
        meses[nasc.month-1] += 1
    plt.bar(range(1,13), meses)
    plt.xlabel("Mês de nascimento")
    plt.ylabel("Quantidade")
    plt.title("Nascimentos por mês")
    plt.show()

def salvar_json(pessoas, nome_sorteado, arquivo="resultado.json"):
    """Salva dados em JSON"""
    with open(arquivo, "w", encoding="utf-8") as f:
        json.dump({"pessoas": pessoas, "nome_sorteado": nome_sorteado}, f, ensure_ascii=False, indent=2)

# Exemplo de uso (assume arquivo CSV com colunas "nome" e "data_nasc" no formato YYYY-MM-DD)
pessoas = ler_csv("pessoas.csv")
pessoas = calcular_idades(pessoas)
nome_sorteado = sorteio_nome(pessoas)
grafico_nascimentos(pessoas)
salvar_json(pessoas, nome_sorteado)
print(f"Nome sorteado: {nome_sorteado}")
