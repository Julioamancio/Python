Sintaxe Básica do Python

Python utiliza indentação (espaço ou tabulação) para definir blocos de código, não usa chaves { } como C ou Java. Isso obriga o código a ser limpo e fácil de ler. O erro mais comum de iniciantes é esquecer a indentação.

Exemplo 1 (correto): 
# Se a condição for verdadeira, executa o bloco indentado
if 5 > 2:
    print("Cinco é maior que dois!")
Exemplo 2 (errado): 
# Aqui falta a indentação na linha do print, o que gera erro!
if 5 > 2:
print("Erro de indentação!")
Exemplo 3 (correto com função): 
# Função indentada corretamente
def saudacao():
    print("Olá, mundo!")
saudacao()
Variáveis e Tipos de Dados

Variáveis armazenam valores em Python. Não é preciso declarar o tipo antes — ele é definido automaticamente pelo valor atribuído.
Principais tipos:

  • int: números inteiros idade = 18
  • float: números decimais altura = 1.75
  • str: textos (strings) nome = "Julio"
  • bool: valores lógicos (True ou False) aprovado = True
Exemplo 1: 
# Variáveis de texto e inteiro
nome = "Julio"
idade = 18
print(nome, idade)
Exemplo 2: 
# Variáveis de texto e decimal
preco = 12.90
produto = "Caderno"
print("Produto:", produto, "- Preço:", preco)
Exemplo 3: 
# Variáveis de tipos diferentes
tem_carteira = False
peso = 72.5
cidade = "Sorocaba"
print(cidade, peso, tem_carteira)
Operadores Aritméticos, Lógicos e Relacionais

Aritméticos: Realizam cálculos matemáticos.
+ soma, - subtração, * multiplicação, / divisão, ** potência, % resto.

Relacionais: Comparam valores e retornam True ou False.
== igual, != diferente, > maior, < menor, >= maior ou igual, <= menor ou igual.

Lógicos: Combinam condições.
and (e), or (ou), not (negação).

Exemplo 1: 
# Operações matemáticas
print(3 + 2 * 4)           # 11
print(10 / 3)              # 3.333...
print(2 ** 3)              # 8
Exemplo 2: 
# Operadores lógicos e relacionais
a = 5
b = 7
print(a > 3 and b < 10)    # True (ambas verdadeiras)
print(a == b or b == 7)    # True (uma é verdadeira)
Exemplo 3: 
# Mais exemplos de comparação
idade = 17
print(idade >= 18)         # False (não é maior de idade)
print(not(idade < 18))     # False (negação)
print(idade != 15)         # True (diferente de 15)
Entrada (input()) e Saída (print())

Para interagir com o usuário:
Entrada: input() lê um texto digitado pelo usuário.
Saída: print() mostra informações na tela.

Exemplo 1: 
# Pede o nome e cumprimenta
nome = input("Seu nome: ")
print("Oi,", nome)
Exemplo 2: 
# Lê idade digitada e mostra
idade = input("Qual sua idade? ")
print("Idade digitada:", idade)
Exemplo 3: 
# Lê dois dados e mostra juntos
animal = input("Animal favorito: ")
cor = input("Cor favorita: ")
print("Você gosta de", animal, "e da cor", cor)
Conversão de Tipos (int(), str(), float())

Ao usar input(), o valor lido é sempre texto (str).
Para fazer contas, converta para int (inteiro) ou float (decimal).
Use str() para converter números em texto.

Exemplo 1: 
# Converta texto em inteiro para somar
idade = int(input("Digite sua idade: "))
print("Ano que vem:", idade + 1)
Exemplo 2: 
# Converta texto em float para calcular média
n1 = float(input("Nota 1: "))
n2 = float(input("Nota 2: "))
media = (n1 + n2)/2
print("Média:", media)
Exemplo 3: 
# Converta número em texto
valor = 7
texto = str(valor)
print("Agora é string:", texto)
Principais Métodos de String

Strings em Python possuem métodos para tratar e analisar textos.
Métodos são funções prontas que você usa com o ponto: texto.método(). Veja os mais comuns:

.strip(): Remove espaços (ou caracteres) do começo e fim da string, útil para limpar dados digitados. 
frase = "  Python  "
print(frase.strip())  # "Python"

nome = "\tJulio\n"
print(nome.strip())   # "Julio"

texto = "!!!olá!!!"
print(texto.strip("!"))  # "olá"
.replace(): Troca todas as ocorrências de um texto por outro. Bom para padronizar ou corrigir textos. 
mensagem = "Eu gosto de Python"
print(mensagem.replace("Python", "pizza"))  # "Eu gosto de pizza"

texto = "banana,maçã,uva"
print(texto.replace(",", " | "))  # "banana | maçã | uva"

print("aaaah!".replace("a", "e"))  # "eeeeh!"
.upper(): Transforma toda a string em MAIÚSCULAS, útil para padronizar textos. 
nome = "julio"
print(nome.upper())  # "JULIO"

frase = "python é legal"
print(frase.upper())  # "PYTHON É LEGAL"

palavra = "AbC123"
print(palavra.upper())  # "ABC123"
.lower(): Transforma toda a string em minúsculas. 
print("PYTHON".lower())  # "python"

print("Python É Top".lower())  # "python é top"

print("AbC123".lower())  # "abc123"
.count(): Conta quantas vezes um texto aparece na string, bom para análise de dados. 
print("banana".count("a"))  # 3

print("abracadabra".count("bra"))  # 2

print("111-222-333".count("-"))  # 2
.find(): Retorna a posição da primeira ocorrência do texto; -1 se não achar. 
print("olá mundo".find("mundo"))  # 4

print("abcdefg".find("d"))  # 3

print("python".find("z"))  # -1
.startswith(): Retorna True se a string começa com o texto dado. 
print("python é top".startswith("py"))  # True

print("banana".startswith("ba"))  # True

print("Python".startswith("P"))  # True
.endswith(): Retorna True se a string termina com o texto dado. 
print("arquivo.txt".endswith(".txt"))  # True

print("python".endswith("on"))  # True

print("banana".endswith("ana"))  # True
Funções Simples (def, return)

Funções agrupam comandos para reutilizar e organizar o código. Use def para criar e return para devolver valores.

Exemplo 1: 
# Função que soma dois números
def soma(a, b):
    return a + b
print(soma(10, 5))
Exemplo 2: 
# Função que retorna o dobro de um número
def dobro(x):
    return x * 2
print(dobro(7))
Exemplo 3: 
# Função que diz se um número é par ou ímpar
def par_ou_impar(n):
    if n % 2 == 0:
        return "Par"
    else:
        return "Ímpar"
print(par_ou_impar(3))
print(par_ou_impar(8))
Comentários (#)

Comentários servem para explicar o código e não são executados. Comece a linha com # para inserir uma observação para você ou outros programadores.

Exemplo 1: 
# Exibe uma mensagem na tela
print("Olá!")
Exemplo 2: 
a = 5  # Variável com valor 5
b = 3
# Soma e mostra o resultado
print(a + b)
Exemplo 3: 
# Função para mostrar oi
def mostra_oi():
    # Só mostra a mensagem
    print("Oi!")
mostra_oi()
Boas Práticas Básicas

Siga boas práticas para que seu código seja mais fácil de entender e manter:

  • Use nomes claros para variáveis (idade, nome_completo).
  • Sempre indente seus blocos de código (use 4 espaços).
  • Comente partes importantes para facilitar a leitura.
  • Evite repetir código: use funções para reaproveitar lógica.
Exemplo 1 (nome claro e identação): 
# Função bem nomeada e indentada corretamente
def calcula_media(nota1, nota2):
    return (nota1 + nota2) / 2
Exemplo 2 (comentando o código): 
# Calcula a área do quadrado
lado = 5
area = lado * lado
print("Área:", area)
Exemplo 3 (evite código repetido): 
# Ruim: repete o print várias vezes
print("Olá, Ana")
print("Olá, Bia")
# Melhor: cria uma função para reutilizar
def ola(nome):
    print("Olá,", nome)
ola("Ana")
ola("Bia")
Desafio Final do Nível Iniciante

Desafio: Crie um programa completo que:

  • Pede o nome e a idade do usuário
  • Diz se é menor ou maior de idade
  • Mostra o nome em letras maiúsculas e a idade daqui a 10 anos
Solução sugerida (comentada): 
# Pede nome e idade ao usuário
nome = input("Digite seu nome: ")
idade = int(input("Digite sua idade: "))

# Verifica se é maior ou menor de idade
if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

# Mostra nome em maiúsculas
print("Seu nome em maiúsculas:", nome.upper())

# Mostra idade daqui a 10 anos
print("Sua idade daqui a 10 anos será:", idade + 10)
